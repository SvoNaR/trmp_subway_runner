# ОТЧЕТ О ВЫПОЛНЕННОЙ РАБОТЕ
## Разработка мобильного приложения "Subway Runner"

---

## 1. КРАТКОЕ ОПИСАНИЕ ПРОЕКТА

Разработано полнофункциональное мобильное приложение в жанре **endless runner** (бесконечный раннер) по типу игры **Subway Surfers** с клиент-серверной архитектурой.

**Основная идея:** Игрок управляет персонажем, который бежит по трем полосам движения. Необходимо уворачиваться от препятствий и собирать монеты. Результаты сохраняются в онлайн-таблицу лидеров.

---

## 2. ЧТО БЫЛО СДЕЛАНО

### 2.1 Серверная часть

✅ **Разработан backend на Node.js:**
- Веб-фреймворк Express.js для обработки HTTP запросов
- REST API со следующими эндпоинтами:
  - `POST /api/register` - регистрация пользователя
  - `POST /api/login` - аутентификация и выдача JWT токена
  - `POST /api/scores` - сохранение результатов игры
  - `GET /api/leaderboard` - получение таблицы лидеров
  - `GET /api/progress` - получение прогресса игрока

✅ **База данных SQLite:**
- Таблица `users` - хранение учетных записей
- Таблица `game_scores` - результаты игровых сессий
- Таблица `user_progress` - статистика и достижения
- Хеширование паролей с помощью bcrypt
- Автоматическое создание таблиц при запуске

✅ **Система безопасности:**
- JWT токены для аутентификации
- Middleware для проверки прав доступа
- Валидация входящих данных
- Защита от SQL инъекций через параметризованные запросы

### 2.2 Клиентская часть

✅ **Мобильное приложение на React Native (Expo):**

**Экран регистрации:**
- Поля: имя пользователя, email, пароль, подтверждение пароля
- Валидация совпадения паролей
- Проверка длины пароля (минимум 6 символов)
- Интеграция с сервером для создания аккаунта

**Экран входа:**
- Аутентификация по email и паролю
- Сохранение JWT токена в AsyncStorage
- Переход на главный экран после успешного входа

**Главное меню:**
- Яркий дизайн с логотипом "SUBWAY RUNNER"
- Кнопки: "ИГРАТЬ", "ТАБЛИЦА ЛИДЕРОВ", "ПРОФИЛЬ"
- Приветствие пользователя

**Игровой экран:**
- **3 полосы движения** с разделительными линиями
- **Управление свайпами** влево/вправо
- **Генерация препятствий** (красные квадраты)
- **Сбор монет** (желтые круги)
- **Подсчет очков**: +1 за расстояние, +10 за монету
- **Постепенное увеличение скорости** каждые 500 метров
- **Определение столкновений** с препятствиями
- **Сохранение результата** после окончания игры

**Таблица лидеров:**
- Загрузка топ-10 игроков с сервера
- Сортировка по максимальному счету
- Pull-to-refresh для обновления данных
- Отображение с медалями 🥇🥈🥉

**Профиль пользователя:**
- Аватар с первой буквой имени
- Статистика: игр, рекорд, монеты, уровень
- Кнопки настроек, достижений, помощи
- Выход из аккаунта

## 6. КАК ЗАПУСТИТЬ ПРОЕКТ

### Быстрый старт:

**1. Запуск сервера:**
```bash
cd SubwayRunner/server
npm install
npm start
```

**2. Запуск приложения (в новом терминале):**
```bash
cd SubwayRunner
npm install
npx expo start
```

**3. Настройка соединения:**
- Узнайте ваш IP: `ipconfig` (Windows) или `ifconfig` (macOS/Linux)
- Замените `192.168.1.129` на ваш IP в файлах:
  - `src/services/api.js`
  - `src/screens/LoginScreen.js`
  - `src/screens/RegisterScreen.js`
  - `src/screens/GameScreen.js`

**4. Запуск на устройстве:**
- Установите Expo Go из Google Play / App Store
- Отсканируйте QR-код из терминала
