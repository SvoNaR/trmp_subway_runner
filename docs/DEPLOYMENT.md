# ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ ПРИЛОЖЕНИЯ "SUBWAY RUNNER"

## 1. Требования к системе

### Для серверной части:
- Node.js версии 14.x или выше
- npm версии 6.x или выше
- Свободное место на диске: минимум 100 МБ

### Для мобильной разработки:
- Expo CLI (последняя версия)
- React Native
- Android Studio (для эмулятора Android) или Xcode (для iOS)
- Устройство Android/iOS с приложением Expo Go (опционально)

---

## 2. Установка и запуск серверной части

### Шаг 1: Перейдите в директорию сервера
```bash
cd SubwayRunner/server
```

### Шаг 2: Установите зависимости
```bash
npm install
```

### Шаг 3: Запустите сервер
```bash
npm start
```

Или в режиме разработки с автоперезагрузкой:
```bash
npm run dev
```

### Шаг 4: Проверка работы сервера
Сервер должен запуститься на порту 3000.  
Проверьте работу API через браузер или Postman:
- http://localhost:3000/api

**Пример запроса для проверки:**
```bash
curl http://localhost:3000/api/leaderboard
```

---

## 3. Установка и запуск клиентской части

### Шаг 1: Откройте новый терминал и перейдите в корень проекта
```bash
cd SubwayRunner
```

### Шаг 2: Установите зависимости (если еще не установлены)
```bash
npm install
```

### Шаг 3: Запустите приложение Expo
```bash
npm start
```

или

```bash
npx expo start
```

### Шаг 4: Выбор способа запуска

#### Вариант A: Запуск на эмуляторе Android
1. Запустите Android эмулятор в Android Studio
2. В терминале нажмите `a` для запуска на Android

#### Вариант B: Запуск на физическом устройстве Android
1. Установите приложение "Expo Go" из Google Play
2. Отсканируйте QR-код, показанный в терминале

#### Вариант C: Запуск на симуляторе iOS (только macOS)
1. В терминале нажмите `i` для запуска на iOS

---

## 4. Настройка сетевого взаимодействия

### Важно! Измените IP-адрес в настройках

В файле `src/services/api.js` и экранах измените IP-адрес на ваш локальный:

```javascript
const API_BASE_URL = 'http://192.168.1.129:3000/api';
```

**Как узнать ваш IP-адрес:**

Windows:
```bash
ipconfig
```

macOS/Linux:
```bash
ifconfig
```

Используйте IPv4-адрес вашей сети.

## 5. Развертывание в магазине приложений

### Публикация в Google Play Store

#### Шаг 1: Создание APK файла
```bash
eas build --platform android
```

#### Шаг 2: Настройка EAS Build
```bash
eas build:configure
```

#### Шаг 3: Сборка APK
```bash
eas build -p android --profile preview
```

#### Шаг 4: Публикация
1. Создайте аккаунт разработчика в Google Play Console ($25 единоразово)
2. Загрузите APK файл
3. Заполните информацию о приложении
4. Отправьте на модерацию

### Публикация в Apple App Store

#### Шаг 1: Создание IPA файла
```bash
eas build --platform ios
```

#### Шаг 2: Настройка сертификатов
Необходим аккаунт Apple Developer ($99/год)

#### Шаг 3: Сборка
```bash
eas build -p ios --profile production
```

#### Шаг 4: Публикация через App Store Connect
1. Войдите в App Store Connect
2. Создайте новое приложение
3. Загрузите IPA через Transporter
4. Заполните метаданные
5. Отправьте на ревью

## 7. Устранение неполадок

### Ошибка: "Network request failed"
**Решение:**
- Проверьте, что сервер запущен
- Убедитесь, что IP-адрес в настройках верный
- Проверьте, что устройства в одной сети

### Ошибка: "Port 3000 is already in use"
**Решение:**
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# macOS/Linux
lsof -ti:3000 | xargs kill -9
```

### Ошибка: "Module not found"
**Решение:**
```bash
npm install
```

### Приложение не запускается на устройстве
**Решение:**
- Обновите Expo Go до последней версии
- Проверьте версию React Native в проекте
- Очистите кэш: `npx expo start -c`
