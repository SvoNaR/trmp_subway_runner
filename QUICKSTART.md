# БЫСТРЫЙ СТАРТ - SUBWAY RUNNER

## ⚡ Быстрый запуск приложения

### Шаг 1: Установка зависимостей

**Установите зависимости для сервера:**
```bash
cd SubwayRunner\server
npm install
```

**Установите зависимости для клиента (в новом терминале):**
```bash
cd SubwayRunner
npm install
```

### Шаг 2: Запуск сервера

**В терминале 1 (сервер):**
```bash
cd SubwayRunner\server
npm start
```

✅ Сервер запустится на порту **3000**  
✅ База данных создастся автоматически  
✅ Вы увидите: `Server running on http://localhost:3000`

### Шаг 3: Настройка IP-адреса

**Узнайте ваш IPv4 адрес:**

Windows (PowerShell):
```powershell
ipconfig
```

Найдите строку **IPv4 Address** (например, `192.168.1.129`)

**Замените IP в файлах:**

Откройте файл `src/services/api.js` и замените:
```javascript
const API_BASE_URL = 'http://ВАШ_IP:3000/api';
```

Например:
```javascript
const API_BASE_URL = 'http://192.168.1.129:3000/api';
```

**Сделайте то же самое в файлах:**
- `src/screens/LoginScreen.js` (строка ~28)
- `src/screens/RegisterScreen.js` (строка ~46)
- `src/screens/GameScreen.js` (строка ~227)

### Шаг 4: Запуск мобильного приложения

**В терминале 2 (клиент):**
```bash
cd SubwayRunner
npx expo start
```

После запуска вы увидите QR-код.

### Шаг 5: Запуск на устройстве

**Вариант A: Эмулятор Android**
1. Запустите Android эмулятор в Android Studio
2. Нажмите клавишу `a` в терминале Expo

**Вариант B: Физическое устройство Android**
1. Установите приложение **Expo Go** из Google Play
2. Отсканируйте QR-код из терминала

**Вариант C: Симулятор iOS (только macOS)**
1. Нажмите клавишу `i` в терминале Expo

## 🐛 Решение проблем

### Ошибка: "Network request failed"

**Причина:** Неправильный IP или устройства в разных сетях

**Решение:**
1. Проверьте, что сервер запущен
2. Узнайте правильный IP через `ipconfig`
3. Обновите IP во всех файлах
4. Убедитесь, что телефон и компьютер в одной WiFi сети

### Ошибка: "AsyncStorage.slice is not a function"

**Решение:**
```bash
npm install @react-native-async-storage/async-storage
```

Затем перезапустите приложение:
```bash
npx expo start -c
```

### Ошибка: "Port 3000 is already in use"

**Windows:**
```powershell
netstat -ano | findstr :3000
taskkill /PID <PID> /F
```

**Или измените порт в `server/server.js`:**
```javascript
const PORT = 3001; // вместо 3000
```

### Приложение не запускается

**Очистите кэш:**
```bash
npx expo start -c
```

### Экран регистрации не работает

**Проверьте:**
1. Сервер запущен
2. IP-адрес правильный
3. Интернет соединение есть
4. Брандмауэр не блокирует порт 3000

## 📱 Как играть

1. **Управление:**
   - Свайп влево ← перемещение влево
   - Свайп вправо → перемещение вправо

2. **Цель:**
   - Бежать как можно дальше
   - Собирать монеты (+10 очков)
   - Избегать красных препятствий

3. **Очки:**
   - +1 очко за каждый метр
   - +10 очков за каждую монету

4. **Конец игры:**
